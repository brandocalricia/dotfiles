## 2. dotfiles auto-sync + tracked per-host config  (2026-07-14)
A **user** timer `dotfiles-sync.timer` auto pull-rebase/commit/pushes
`~/dotfiles` every 15 min, so config changes propagate between machines on their
own. Per-host configs are namespaced `local.<host>.*`.
Install (user scope, no sudo):
```
cp ~/dotfiles/systemd/dotfiles-sync.service ~/dotfiles/systemd/dotfiles-sync.timer ~/.config/systemd/user/
systemctl --user daemon-reload
systemctl --user enable --now dotfiles-sync.timer
bash ~/dotfiles/scripts/apply-host-local.sh
```
**Verify:** `systemctl --user status dotfiles-sync.timer` active;
`systemctl --user list-timers | grep dotfiles`. Detail: CLAUDE.md.
